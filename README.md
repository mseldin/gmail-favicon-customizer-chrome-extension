# gmail-favicon-customizer-chrome-extension
Chrome plugin to customize gmail favicon tabs in chrome (including unread messages indicator)

This page lives at https://github.com/mseldin/gmail-favicon-customizer-chrome-extension/edit/master/README.md

The original source was for a now defunct chrome extension available here: https://chrome.google.com/webstore/detail/gmail-favicon-customizer/naelggiioddkpaokkopkbgfcjlkldpdn

This version does not attempt to display unread message counts.

Otherwise, only trivial changes have been made from ecaroths original at https://github.com/ecaroth/gmail-favicon-customizer-chrome-extension, basically to simplify it a bit and make it use a v3 manifest.
